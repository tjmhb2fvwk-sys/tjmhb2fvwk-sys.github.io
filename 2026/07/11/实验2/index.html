实验报告2：代码编辑与管理

实验报告：代码编辑与管理

1、实验目的

安装和使用文本代码编辑器完成 Markdown 的编辑和渲染，使用 Git 进行版本管理等。了解和实践代码编辑器的配置、Markdown 语法、版本管理工具的使用等。

2、实验环境

  • 操作系统：Windows 11 专业版 22H2
  • 软件版本：Git 2.55.0，VS Code 1.128.0
  • VS Code 插件：Markdown All in One、Markdown Preview Enhanced、Paste Image

3、实验过程

步骤一：安装 Git for Windows

访问 Git 官网（https://git-scm.com/download/win）下载并运行安装包。在 “Select Components” 页面勾选 “Git Bash Here” 和 “Git GUI Here”，在 “Adjusting your PATH environment” 页面选择 “Git from the command line and also from 3rd-party software”。安装完成后重启命令提示符，执行 git --version 验证安装。


图1：Git for Windows 安装完成界面


图2：Git 版本验证与用户信息配置

步骤二：安装 Visual Studio Code

访问 VS Code 官网（https://code.visualstudio.com/）下载并运行安装包。在 “选择附加任务” 页面务必勾选 “创建桌面快捷方式”、”将Code添加到资源管理器上下文菜单” 和 “添加到PATH”。启动 VS Code 后熟悉主界面布局。


图3：VS Code 安装完成界面


图4：VS Code 主界面布局

步骤三：安装 Markdown 相关插件

点击左侧活动栏中的扩展图标（或使用 Ctrl + Shift + X），搜索并安装 Markdown All in One、Markdown Preview Enhanced 和 Paste Image 插件。安装完成后，配置 files.autoSave 为 afterDelay，editor.wordWrap 为 on。


图5：已安装的 Markdown 相关插件列表

步骤四：创建并编辑第一个 Markdown 文件

新建文件 test.md，输入 Markdown 内容。使用 Ctrl + K V 并排预览，验证 Markdown 语法渲染效果。


图6：Markdown 源码与预览分屏模式

步骤五：Git 版本管理

在项目目录下执行 git init 初始化 Git 仓库，创建 .gitignore 文件并添加 node_modules/、public/ 等忽略项。执行 git add . 和 git commit -m "初始化Hexo博客项目" 提交代码。使用 git log 和 git status 查看提交历史和当前状态。

git status 和 git log 命令输出截图
图7：Git 版本管理状态与提交历史

4、实验结果

测试项 目标 结果
Git 安装 git –version ✅ 2.42.0
Git 配置 git config –global user.name ✅ 已配置
VS Code 安装 启动正常 ✅ 成功
Markdown 预览 分屏渲染 ✅ 成功
Git 初始化 git init ✅ 成功
代码提交 git commit ✅ 成功

5、知识总结

命令 作用
git init 初始化 Git 仓库
git add . 添加所有更改到暂存区
git commit -m "信息" 提交更改
git log 查看提交历史
git status 查看当前状态

6、出现问题

问题1：安装 VS Code 后无法在命令行使用 code 命令。

  • 原因：安装时未勾选 “添加到 PATH”。
  • 解决：重新运行安装程序，勾选 “添加到 PATH”；或手动将 VS Code 安装目录添加到系统环境变量。

问题2：[TOC] 不生成目录。

  • 原因：未安装 Markdown All in One 插件。
  • 解决：安装并启用 Markdown All in One 插件。

7、心得体会

通过本次实验，我掌握了 VS Code 这一轻量级代码编辑器的安装、配置和插件管理方法。Markdown 插件极大地提升了文档编写效率，让我能够专注于内容创作而非排版细节。Git 版本管理让我体会到了代码追踪和协作的重要性，为后续的项目开发奠定了良好基础。同时，Git 作为开源社区的核心工具，其分布式版本控制理念体现了开源协作的智慧。【课程大纲-思政融合点4】
